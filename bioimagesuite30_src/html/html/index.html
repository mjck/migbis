BioImage Suite Command Line Tools Index

Note: Any tool can be executed also as an applet with a graphical user interface using the --dogui 1 flag.

For example in the case of bis_smoothimage, you can invoke this as an applet using:

bis_smoothimage.tcl --dogui 1

Image Processing

bis_castimage , cast (or convert) image data type.
bis_clusterthresholdimage , thresholds an image to generate a binary output using a specific threshold and clusters to a certain voxel size.
bis_convolveimage , Convolvees an image with a specific gaussian/gaussian derivative/hessian kernel.
bis_cropimage , Crops an image with a specific cropping window/radius.
bis_cylindricalcropimage , Blanks an image with a specific cylinder.
bis_distancemapimage , Create a distance map from a binarized image
bis_edgedetectimage , Computes an edge map for an image following smoothing
bis_fftimage , computes the fourier transformation of an image.
bis_flipimage , flips image about one of the coordinate axes.
bis_imagedilate , Modifies an image using continuous dilation filtering
bis_imagemedian , Smoothes an image using median filtering
bis_listthresholdimage , thresholds an image to generate a binary output using a specific threshold.
bis_logimage , computes linear intensity transformation
bis_newreorientimage , newreorients an image to a given orientation.
bis_nonlinearsmoothimage , Smoothes an image using anisotropic diffusion filtering
bis_piecewiseimagemap , Perfoms piecewise mapping of an image using a linear spline with prescribed knots.
bis_proportionalscale , Proportional scales input images to the mean specified in the options.
bis_regularizeobjectmap , Regularizes an objectmap (an image where each value represents an object) to smooth manual segmentations
bis_relabelimage , relabels an image to a given orientation. Output orientation is 0=axial, 1=coronal, or 2=sagittal.
bis_reorientimage , reorients an image to a given orientation.
bis_resampleimage , Resamples an image to a specific resolution with optional blurring.
bis_rfftimage , computes inverse fourier transform.
bis_shiftscaleimage , computes linear intensity transformation
bis_skeletonizeimage , Create a skeleton and/or a distance map from a binarized image. if dodistance is 1 multiples by inverse distance map
bis_smoothimage , Smoothes an image with a specific gaussian kernel.
bis_thresholdimage , thresholds an image to generate a binary output using a specific threshold.

bis_imageremoveframes , Extracts frames from a 4D image

bis_mediantemporalsmoothimage , Smooth an image with median filter.
bis_preprocessimage , Preprocess an image.
bis_removeslicemean , Remove Slice/Volumn Mean.
bis_removetemporaldrift , Correct the drift of an image.
bis_temporalsmoothimage , Temporal smoothing with Gaussian kernel.

bis_imagemathoperations , Add, Subtract, Multiply or Divide two images to give an output.
bis_manualreslice , reslices an image using one or more transformations.
bis_maskimage , blends an image using one or more transformations.
bis_resliceimage , reslices an image using one or more transformations.
bis_roimean , Calculate the mean time course of each ROI defined by ROI mask.
bis_singleimagemathoperations , Image to Image algorithm that does more complex operations than bis_imagemathoperations
bis_sliceinhomogeneity , corrects slice inhomogeneity correction for MRI acquisitions.
bis_stitchimage , Stitches images

Registration

bis_combinetransformations , computes a combined (optionally inverse) transformation for a set transformation given an image to define the space.
bis_computedisplacementfield , computes the displacement field for a transformation given an image to define the space.
bis_computeoverlap , computes overlap metrics of the pre-segmented structures after registration.
bis_computesimilarity , computes image similarity measures between the reference image and the target image after a transform and optionally point distances between the reference surface and the target surface.
bis_linearintensityregister , computes a linear intensity based registrations.
bis_linearpointregister , computes a linear point based registrations.
bis_manualregistration , Computes a manual registration of an image
bis_nonlineardistortioncorrection , Computes non-linear distortion corrections for echoplanar images.
bis_nonlinearintensityregister , computes a linear intensity based registrations.
bis_nonlinearpointregister , computes a linear point-based registrations.
bis_serialdemonmousereg , Serial demon registration
bis_singlejacobian , computes either the determinant of the jacobian or the full tensor for a transformation.
bis_visualizetransformation , Creates a grid visualization of a transformation

bis_computecorrelations , Compute Correlations between the 4D image as input 1 and the measures in filename. 4D Image must have the same number of frames as the subjects in the filename.
bis_computeglm , Compute GLM Fit using Matrix to define regressors. Optionally save the residual image.
bis_fmritoolbox , fmri Tool Box. Calculate complex network measures.
bis_intrinsicconnectivity , Calculate the intrinsic connectivity of an image.
bis_matrixcorrelation , Calculate correlation among time series.
bis_multisubjectfmri , Mostly a test class for pxitclmultisubject.tcl based multi subject operations
bis_rtopvalue , Converts r value to 1-p value.
bis_rtotmap , convert r value to t value.
bis_seedcorrelation , Correlation between brain and ROIs.
bis_singlesubjectfmri , Mostly a test class for bis_fmrisetup.tcl based single subject fMRI Operations
bis_tmaptopvalue , converts tmap to pvalue.
bis_tmaptozscore , converts tmap to zscore.
bis_zscoretopvalue , Converts zmap to pvalue.
